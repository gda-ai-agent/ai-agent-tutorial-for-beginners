## 🚀 快速開始

### 第一步：安裝 OpenClaw

```bash
# macOS / Linux
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows (PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### 第二步：初始化配置

```bash
openclaw onboard
```

### 第三步：開始使用

```bash
# 查看幫助
openclaw --help

# 查看狀態
openclaw status
```
